## 1. How I track progress
This project is developed iteratively. Each lab checkpoint is documented using:

1) What did I complete?  
2) What’s my next step?  
3) Any blockers?

## 2. Weekly log (fill this in as you go)
| Week | What I completed | Next step | Blockers / risks | Evidence (commit / screenshot) |
|---|---|---|---|---|
| W1 | Grid created + start/goal + printing | Research heuristics | Heuristic choice | link |
| W2 | Manhattan heuristic integrated | Implement A* open/closed sets | PQ ordering | link |
| W3 | Path reconstruction + visual output | Expand tests + edge cases | none | link |
| W4 | Stats (path length, expanded) | Improve report visuals | time | link |
